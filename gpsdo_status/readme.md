# AERPAW GPSDO Status Query Tool

This tool allows to query the status of the custom AERPAW GPSDO module.

## Install

Create a build directy and cd into it.

```
mkdir build
cd build
```

Create the build files using CMake.

```
cmake .. -DCMAKE_BUILD_TYPE=Release
```

### Windows

Build and install the software using cmake.

```
cmake --build . --config Release
cmake --install .
```

### Linux / MacOS

Build and install the software using make.

```
make
sudo make install
```
